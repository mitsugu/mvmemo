## Install
1. git clone https://github.com/mitsugu/mvmemo.git
2. cd mvmemo
3. chmod +x mv
4. cp mv ~/.config/memo/plugins <- Specify the directory of the pluginsdir entry in config.toml
5. cd ..
6. rm -fr mvmemo

## Usage
memo mv SourceFile DestnationFile
